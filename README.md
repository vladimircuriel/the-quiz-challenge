**The Quiz Challenge Analytics | Data Mining Model** is a data analytics project focused on building an interactive dashboard and a descriptive data mining model using YouTube channel data. It implements an ETL pipeline to extract statistics from YouTube Analytics and the YouTube API, transform the data with Tableau Prep, and visualize insights in Tableau to analyze content performance, audience behavior, and engagement patterns.
